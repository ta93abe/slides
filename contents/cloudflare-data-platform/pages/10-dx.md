<div v-click class="mt-4 text-sm op-70">

**vs AWS CLI / SAM / CDK**: AWS は CLI・SAM・CDK・CloudFormation と選択肢が多いが分散している。Wrangler は1つの CLI に統合。ローカル開発も Miniflare は本番同一ランタイム（workerd）で動くが、LocalStack は AWS API のエミュレーションで差異が出やすい。ただし AWS のIaCエコシステム（CDK, CloudFormation）の成熟度は圧倒的。
