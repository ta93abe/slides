# データエンジニアの日常

```mermaid
graph LR
    S3[S3] -->|エグレス $0.09/GB| Snowflake[Snowflake]
    Snowflake -->|エグレス| BI[BI ツール]
    S3 -->|エグレス $0.09/GB| DuckDB[手元の DuckDB]

```

100GB のデータを手元の DuckDB で分析したい

→ S3 エグレス: **$9**（毎回）

→ 毎日やったら月 **$270**（データ見るだけで）

---

# エグレス料金の不条理

| やりたいこと | AWS のコスト |
|---|---|
| S3 のデータを Athena でクエリ | スキャン料 + エグレス |
| S3 → Snowflake にロード | エグレス |
| S3 → 手元の DuckDB で分析 | エグレス |
| S3 → 別リージョンにコピー | **リージョン間転送料** |
| S3 のデータを誰かに共有 | エグレス |

> データを**置くのは安い**、**出すのが高い**
>
> → これが **Cloud の罠** であり **ロックイン** の本質
