# AI Search（AutoRAG）

キーワード検索 + AI セマンティック検索のマネージド検索サービス。

<v-click>

**データカタログの検索エンジン**として活用:

</v-click>

<v-click>

```
dbt artifacts (manifest.json / catalog.json)
  → R2 に保存
    → Worker がカタログドキュメント生成
      → AI Search に自動インデキシング
        → 「売上に関連するテーブルは？」で検索
```

</v-click>

| カスタマイズ性 | 低 | 高 |
