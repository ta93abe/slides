# Cloudflare Data Platform の強み

<v-clicks>

1. **エグレス $0** — データを R2 に集めれば、どのエンジンからも無料で読める
2. **Apache Iceberg 標準** — ベンダーロックインなし、いつでも脱出可能
3. **サーバーレス** — インフラ管理ゼロ、使った分だけ課金
4. **エッジで完結** — インジェストからクエリ・配信までワンプラットフォーム
5. **AI ネイティブ** — Workers AI / Agents / AI Search がデータ基盤と直結

</v-clicks>
